## 📱 WhatsApp Integration Options (Researched)

### Free / Open Source
| Option | Cost | Best For |
|--------|------|----------|
| **Evolution API** | Free (self-hosted) | Best overall - active community, n8n integration |
| **BlueBubbles** | Free | iMessage bridging, Mac required |
| **Meta Cloud API** | Free access + conversation fees | Cheapest ongoing, requires dev work |

### Paid Options
| Provider | Price |
|----------|-------|
| Wati | ~$99/mo for 5 users |
| 360dialog | ~$15-50/mo |
| Twilio | Pay-per-message |
| Maton | Check maton.ai/pricing |

**Recommendation:** Evolution API for cost-effective, scalable solution.
